# eslint-config-node

A personal ESLint configuration for Node.js

## Usage

Add this package to your project as a dev-dependency:

```bash
npm install -D @sqrtthree/eslint-config-node
```

Then initialize the linter.

```json
{
  "extends": ["@sqrtthree/eslint-config-node"]
}
```
